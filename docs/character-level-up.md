# Subida de nível local

No editor de personagem, selecione **Subir de nível**. O fluxo usa os dados XPHB da classe e subclasse registradas na ficha para identificar features do próximo nível.

A operação solicita o ganho de pontos de vida, apresenta as features e escolhas pendentes, atualiza referências do compêndio e só grava após confirmação.

Escopo atual: uma classe principal, níveis 1 a 20 e progressões XPHB. Multiclasse, reespecificação e escolhas cuja regra não é estruturada continuam exigindo edição explícita.

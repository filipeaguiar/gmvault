### Descrição e Elementos Importantes

Os vendedores do Mercado Noturno Dyn Singh sabem muito sobre o funcionamento interno do mercado, seus colegas de barraca e as fofocas diárias. Embora sejam tipicamente educados, eles compartilham pouca informação com estranhos. Se os personagens ganharem renome ao vencer os Jogos do Mercado, os seguintes vendedores compartilham os fatos para os valores de renome iguais (e inferiores) ao renome do personagem que estiver conversando com eles. Veja a seção "**Área Gaining Renown**" para detalhes adicionais.


### Vi Aroon

O gêmeo de Kasem e co-proprietário do Spicy Brothers, Vi Aroon, ama o que faz. Ele não sabe nada sobre os planos de Kasem, mas pode ser convencido a compartilhar as fofocas na tabela What Vi Aroon Knows.

**Tabela: What Vi Aroon Knows**

|  |  |
| --- | --- |
| 0 | Vi compartilha que ele e Kasem tiveram uma briga mais cedo, mas não revela o porquê. |
| 1 | Vi explica que está se casando e se mudando para uma terra distante onde sua noiva vive. |
| 2 | Vi confidencia que Kasem está bravo com o casamento iminente de Vi. Kasem disse a Vi que ele está cometendo um erro ao deixar os negócios da família. |
| 3 | Depois que Vi contou a Kasem que estava se casando, Kasem levou vários dias a mais para fazer a corrida de suprimentos regular. Ele voltou com caixotes extras que guarda na tenda privada dos Spicy Brothers. |


### Madam Kulp

Madam Kulp é uma costureira gnoma que trabalha com seda de lagartas tut-krogh. Ela pode compartilhar as fofocas na tabela What Madam Kulp Knows.

**Tabela: What Madam Kulp Knows**

|  |  |
| --- | --- |
| 0 | Madam Kulp faz um comentário direto sobre o comportamento ou aparência dos personagens. |
| 1 | Madam Kulp diz que suas lagartas tut-krogh ficam nervosas sempre que ela as leva para passear perto das barracas Tyenmo ou Xungoon. |
| 2 | Madam Kulp acredita que Lamai e Kusa nunca se machucariam intencionalmente, devido ao dano que a rivalidade de suas famílias causou aos seus pais. |
| 3 | Madam Kulp acha que Kasem tem algum tipo de animal de estimação estranho. Ela o viu alimentando laranjas a uma criatura estranha, parecida com um macaco, mas seus olhos não são tão bons. |

### Sid Squid

Sid Squid é dono do Trusty Leek, uma barraca perto de qualquer loja que contratou os personagens. Eles são o chefe de uma das famílias mais novas do mercado—um bando de aventureiros que se tornaram donos de negócios. Sid pode compartilhar as fofocas na tabela What Sid Squid Knows.

**Tabela: What Sid Squid Knows**

|  |  |
| --- | --- |
| 0 | Sid muda de assunto para contar uma história de seus dias de aventureiro. |
| 1 | Sid explica que alguns vendedores gostam de ver as famílias Xungoon e Tyenmo brigarem. Se as duas se derem bem, isso pode prejudicar outros vendedores que têm negócios em andamento com ambas as famílias. |
| 2 | Sid ouviu Kasem falando sobre comprar um negócio de família para administrar, o que é contra a tradição do mercado. |
| 3 | Sid viu Kasem esconder algo laranja debaixo da barraca da família Xungoon logo antes de uma sabotagem ocorrer. |


### Outros Vendedores

Muitos vendedores dentro do Mercado Noturno Dyn Singh têm conhecimento geral sobre as famílias Tyenmo e Xungoon, bem como sobre os furtos recentes no mercado. Esses vendedores podem compartilhar as fofocas na tabela What Other Vendors Know.

**Tabela: What Other Vendors Know**

|  |  |
| --- | --- |
| 0 | O vendedor muda de assunto, direcionando a atenção dos personagens para suas mercadorias. |
| 1 | O vendedor nota que as perturbações nos negócios Tyenmo e Xungoon causam bastante comoção. |
| 2 | O vendedor diz que algumas pessoas acreditam que as lojas Tyenmo e Xungoon são assombradas. Transeuntes afirmam ter tropeçado quando não havia nada em seu caminho. |
| 3 | O vendedor descreve ter visto um macaco azul perto da tenda Tyenmo ou Xungoon. |
